# Pomodoro Timer ⏱️

A simple and fun Pomodoro timer written in Python. It helps you stay focused with work sessions, take refreshing breaks, and even plays a relaxing rain sound during breaks.

## Features

- Typing-effect intro message
- Customizable work and break durations
- Multiple Pomodoro sessions
- Countdown display in HH:MM:SS format
- Add relaxing sound in break session
